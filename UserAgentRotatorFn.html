UserAgentRotatorFn

UserAgentRotatorFn

Clase que permite rotar y filtrar user agents según su tipo. Útil para realizar solicitudes HTTP con diferentes user agents para evitar bloqueos o simular distintos dispositivos/navegadores.

Constructor

new UserAgentRotatorFn(typeopt)

Description:

  • Crea una nueva instancia del rotador de user agents.

Source:
  • userAgentRotator-fn.js, line 28
Example
const rotator = new UserAgentRotator('mobile');
console.log(rotator.getNext()); // Devuelve el siguiente user agent móvil
console.log(rotator.getRandom()); // Devuelve un user agent móvil aleatorio

rotator.addUserAgent('Mozilla/5.0 (TestBot)');
rotator.setType('bot'); // Cambia a filtrar solo bots
console.log(rotator.getNext()); // Devuelve un user agent de bot
